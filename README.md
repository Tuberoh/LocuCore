# Intro
LocuCore is a management plugin designed for saving, managing, and teleporting to user‑saved locations. The experience is enhanced by an interactive and easy‑to‑understand GUI.

# Commands

- Help Command
```
 /luc help 
```
Shows the list of commands

- Set Command
```
 /luc set <name> 
```
Useful if you want to set your exact position in that moment

```
/luc set <name> <x> <y> <z>
```
Universal set command
- Remove Command

```
/luc remove <name>
```
Remove a specific location

- Tp Command

```
/luc tp <player_name> <location_name>
```
Teleport a specific player to a location

- Modify Command
```
/luc modify <location_name> coordinates 
```
Useful if you want to change the coordinates with your exact position in that moment

```
/luc modify <location_name> coordinates <x> <y> <z>
```
Universal command to modify the coordinates of a specific location
```
/luc modify <location_name> owner <player_name>
```
Useful if you want to pass the property of the location to another player

- Menu Command

```
/luc menu
```
It opens a useful GUI to simplify the previous commands
